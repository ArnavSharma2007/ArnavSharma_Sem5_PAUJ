## What is Java?  
Java requires that you install the JDK (Java Devlopment Kit).  
The JDK contains JRE (Java Runtime Env) as well as the JVM (Java Virtual Machine).  
<i>JVM</i> compiles our human readable code into byte/machine-code, allocates memory and throws exceptions.  
<i>JRE</i> contains the different libraries that our code would be using.   
<hr>

Variables are containers that can hold some value.  
Whether they hold the memory permanently or temporarily depends on data type of the variable (Non-Primitive or Primitive).

<hr>

Operators used include conditional, assignment, logical and arithmetic.

<hr>

Stack Overflow and Stack Underflow are the two terms that can occur when conversion between two data types such as long and int.
Classes that were present in the original JDK version and were carried onto JDK 1.2 are called legacy classes.  
Wrapper classes allow us to create objects, that is char for Character, int for Integer etc.
Conversion from original class to wrapper class is called unboxing, whereas the vice versa is called autoboxing.
