## Conditional Statements

Example Code:  

import java.util.Scanner;  

public class classCode{\
    public static void main(String[] args){\
        final int pin = 1234;\
        System.out.println("My pin is: " + pin);\
        Scanner sc = new Scanner(System.in);\
        int attempts = 0;\
        while (attempts < 3){\
            System.out.print("Enter your pin: ");\
            int pinInput = sc.nextInt();\
            if (pin == pinInput) { \
                System.out.println("Logged in."); \
                return; \
            }\
            else {   
                System.out.println("You entered the wrong pin!"); \
                attempts++;  
            }
        }
    }
}

<hr>

There are two ways to solve the problem in this code:  
1. Code Reusability (Inheritance, Polymorphism)
2. Data Hiding (Encapsulation, Abstraction)

Encapsulation is grouping data/instructions inside of a single entity such as classes. It involves two steps.
- Convert data members to private
- Generate Getters and Setters for the different data members.

### Class, Objects, Functions
- Classes are like templates for our objects.
- Constructors are specialized functions that are called during the initialization of the object.
- Objects are used to store our data members.
- Functions are reusable codes that perform some functionality.
- The new keyword allows us to create the storage for our objects.
